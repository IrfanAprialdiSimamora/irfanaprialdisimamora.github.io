About Me

Halo perkenalkan nama saya Irfan Aprialdi Simamora. Saat ini saya berusia 22 tahun, saya adalah lulusan Teknik Informatika dari Umiversitas Sulawesi Barat. Saya pernah magang di YOY Manajemen sebagai web developer. Saya sangat senang coding karena dengan coding saya bisa membuat aplikasi yang dapat memprmudah pekerjaan manusia. Sekarang saya sedang mengembangkan kemampuan android programming saya. Saat ini saya bekerja sebagai guru di SMK Permata Harapan Batam.

Saya memiliki kemampuan di bidang programming terutama web programming, selain itu saya juga sangat senang menggunakan Microsoft Office Tools.
